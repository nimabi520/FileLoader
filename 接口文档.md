Formal Review System 接口文档
基础信息
Base URL: /api
认证方式: CAS SSO
统一响应格式: Result (code, msg, data)
一、认证模块 (AuthController)
1.1 查询登录状态
接口: GET /api/auth/login
功能: 返回当前登录用户信息
响应示例:
{
"code": 200,
"msg": "操作成功",
"data": {
"loggedIn": true,
"userId": "用户ID",
"name": "用户姓名",
"role": 0,
"sessionId": "Session ID",
"loginTime": "2026-01-01T10:00:00"
}
}
1.2 退出登录
接口: POST /api/auth/logout
功能: 退出登录
二、文件上传模块 (FileUploadController)
2.1 单文件上传
接口: POST /api/file/upload
Content-Type: multipart/form-data
请求参数:
┌──────────┬────────┬──────┬───────────────────┐
│ 参数 │ 类型 │ 必填 │ 描述 │
├──────────┼────────┼──────┼───────────────────┤
│ file │ File │ 是 │ PDF文件(最大20MB) │
├──────────┼────────┼──────┼───────────────────┤
│ fileName │ String │ 是 │ 文件名 │
└──────────┴────────┴──────┴───────────────────┘
响应示例:
{
"code": 200,
"msg": "操作成功",
"data": {
"fileId": "文件唯一ID",
"uploadTime": "2026-01-01T10:00:00",
"batchId": "批次ID"
}
}
2.2 批量上传
接口: POST /api/file/batchUpload
请求参数: file, fileName, batchId(必填)
2.3 下载文件
接口: GET /api/file/{fileId}/download
功能: 下载原始PDF文件
2.4 查询处理队列位置
接口: GET /api/file/{fileId}/query
功能: 查询文件在异步处理队列中的位置
响应示例:
{
"code": 200,
"msg": "操作成功",
"data": {
"fileId": "文件ID",
"position": 2,
"total": 5
}
}
2.5 检查数据校验结果
接口: GET /api/file/{fileId}/check
功能: 获取文件Excel数据校验结果
响应示例:
{
"code": 200,
"msg": "操作成功",
"data": {
"fileId": "文件ID",
"summaryList": [{"project": "项目", "value": "值"}],
"errorList": [{"ruleId": "规则ID", "field": "字段", "currentValue": "当前值",
"errorDesc": "错误描述"}],
"warnList": [{"description": "说明"}]
}
}
2.6 批量结果导出
接口: GET /api/file/{batchId}/batchResult
功能: 导出批次处理结果Excel文件
三、文件记录模块 (FileRecordsController)
3.1 分页查询文件记录
接口: GET /api/file/records
请求参数:
┌──────────┬─────────┬──────┬────────┬───────────────────┐
│ 参数 │ 类型 │ 必填 │ 默认值 │ 描述 │
├──────────┼─────────┼──────┼────────┼───────────────────┤
│ pageNum │ Integer │ 是 │ 1 │ 页码(最小1) │
├──────────┼─────────┼──────┼────────┼───────────────────┤
│ pageSize │ Integer │ 是 │ 10 │ 每页条数(最大100) │
└──────────┴─────────┴──────┴────────┴───────────────────┘
响应示例:
{
"code": 200,
"msg": "操作成功",
"data": {
"pageNum": 1,
"pageSize": 10,
"total": 100,
"pages": 10,
"list": [
{
"fileId": "文件ID",
"fileName": "文件名.pdf",
"uploadTime": "2026-01-01T10:00:00",
"status": "0",
"batchId": "批次ID"
}
]
}
}
3.2 查询单个文件状态
接口: GET /api/file/{fileId}/status
响应示例:
{
"code": 200,
"msg": "操作成功",
"data": {
"status": "0",
"errorMsg": ""
}
}
3.3 查询批次状态
接口: GET /api/file/{batchId}/batchStatus
响应示例:
{
"code": 200,
"msg": "查询批次状态成功",
"data": {
"batchId": "批次ID",
"successFiles": 5,
"failureFiles": 1,
"unprocessedFiles": 2,
"totalFiles": 8
}
}
3.4 批量下载
接口: GET /api/file/{batchId}/batchDownload
功能: 打包下载批次中所有文件(ZIP)
